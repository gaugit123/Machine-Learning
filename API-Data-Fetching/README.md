# Movie Data Collection Using API

## Overview

This project demonstrates how to collect real-world movie data from an external API using Python. The data is retrieved through API requests, processed from JSON format into a structured Pandas DataFrame, and exported as a CSV dataset for further analysis.

---

## Objectives

* Connect to a REST API using Python.
* Retrieve trending movie information.
* Parse JSON responses.
* Handle paginated API responses.
* Convert raw data into a structured dataset.
* Export data to CSV format.

---

## Technologies Used

* Python
* Requests
* Pandas
* JSON
* Jupyter Notebook

---

## Project Workflow

1. Send requests to the movie API.
2. Authenticate using API headers.
3. Retrieve movie data from multiple pages.
4. Extract relevant fields from JSON responses.
5. Convert data into a Pandas DataFrame.
6. Combine records from multiple pages.
7. Export the final dataset as a CSV file.

---

## Features Extracted

The dataset includes:

* Movie ID
* Title
* Overview
* Popularity
* Release Date
* Vote Average
* Vote Count
